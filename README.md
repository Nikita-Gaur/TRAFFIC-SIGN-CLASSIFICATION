# TRAFFIC-SIGN-CLASSIFICATION
 ## Project Overview 
 This project focuses on classifying traffic sign images using machine learning and deep learning techniques.

 ## Objective 
 To correctly identify the traffic sign from image for intelligent transportation systems.

 ## Technologies Used
 - Python
 - Numpy
 - OpenCV
 - Matplotlib
 - Scikit-learn
 - TensorFlow/ Keras

## Approach
-Image Preprocessing and Resizing 
- Data normalization
- Model training using CNN
- Model evaluation using accuracy

## Results 
The model achieved good accuracy in classifying multiple traffic sign classes.

## Future Scope 
- Improve accuracy using advanced CNN models
- Real- time traffic sign detection
- Web deployment using Flash or FastAPI
